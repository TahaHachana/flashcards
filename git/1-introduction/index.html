Git Version Control System Type

What type of version control system is Git, and what key benefit does this provide?

Git is a distributed version control system (DVCS), meaning every clone has the full project history, enabling offline work and fast local operations.

Configuration File Precedence

Name the three levels of Git configuration file precedence (highest → lowest).

  1. .git/config (repository-level)
  2. ~/.gitconfig (user-level)
  3. /etc/gitconfig (system-level)

Git Storage Method

How does Git store changes differently from many other VCS tools?

Git stores snapshots of the entire project at each commit, not deltas. This makes branching and merging efficient.

Git Setup Components

What are the two main components of a typical Git setup?

  1. Git Server – Hosts remote repositories, the central "source of truth."
  2. Git Client – Local repository interface via CLI or GUI.

Git CLI vs GUI

Why is learning the Git CLI recommended before using a GUI?

The CLI maps directly to Git's native commands and concepts, while GUIs may abstract operations with non-standard terminology.

Git Definitive Nature

What does it mean that Git is "definitive"?

Git requires explicit commands for every action; it never auto‑stages, auto‑commits, or syncs without user instruction.

Git Branching

Why is branching in Git considered lightweight?

A branch is just a pointer to a commit; switching branches simply changes the working directory to the snapshot of that branch.

Check Git Version

How do you check your installed Git version?

git --version

List Git Subcommands

How do you list all available Git subcommands?

git help --all

Double Dash Purpose

What's the purpose of the -- double dash in Git commands?

It separates command options from file paths to avoid ambiguity (e.g., when a file name matches a branch/tag name).

Required Configuration

Which two configuration settings must be set before your first commit?

git config user.name "Your Name"
git config user.email "you@example.com"

Initialize Repository

How do you initialize a new repository with main as the default branch?

git init -b main

Git Add Command

What does git add do?

Stages file changes for the next commit.

Add and Commit Separation

Why does Git separate add and commit into two steps?

For flexibility and to allow atomic commits—grouping related changes together for a clear history.

Commit with Message

How do you commit staged changes with a message?

git commit -m "Message"

View Commit History

How can you view a concise commit history?

git log --oneline

Compare Commits

How can you see the differences between two commits?

git diff <commit1> <commit2>

Remove File

How do you remove a file from the repository and stage the deletion?

git rm <file>

Rename File

How do you rename a tracked file in one command?

git mv oldname newname

Clone Repository

How do you clone a remote repository?

git clone https://host/repo.git
