Linux System Components

What are the four main parts of a complete Linux system?

Linux Kernel, GNU Utilities, User Interface, Application Software.

Kernel Role

What is the primary role of the Linux kernel?

It controls all hardware and software, managing resources, memory, processes, and devices.

Virtual Memory

What is virtual memory in Linux and how is it implemented?

Memory created on disk (swap space) and treated as physical RAM, with data swapped between disk and RAM in pages.

init Process

What is the first process started by the Linux kernel?

init — responsible for starting all other processes.

Init Systems

What are the two popular init systems used in Linux?

SysVinit (script-based) and systemd (configuration file–based).

Device Drivers

How does the Linux kernel manage hardware devices?

Via device drivers, either compiled into the kernel or loaded at runtime as kernel modules.

GNU Utilities

Why are GNU utilities important in a Linux system?

They provide essential tools for file management, text processing, and interacting with the kernel.

CLI vs GUI

What is the difference between a CLI and a GUI in Linux?

CLI offers text-based, scriptable control; GUI provides a graphical environment (e.g., GNOME, KDE).

Many Distributions

Why are there so many Linux distributions?

Different communities and goals lead to variations in software choices, interfaces, and package managers.

Core Distributions

Name three core Linux distributions that form the base for others.

Debian, Red Hat, and Slackware.
