Control Flow Purposes

What are the two main purposes of control flow in Rust?

To run code conditionally (if expressions) and to run code repeatedly (loops).

fn main() {
    let condition = true;
    if condition {
        println!("Runs conditionally");
    }
    for _ in 0..3 {
        println!("Runs repeatedly");
    }
}

If Condition Type

What must the condition in an if expression evaluate to in Rust?

A bool. Rust does not automatically convert non-Boolean values.

fn main() {
    let number = 3;
    if number != 0 {
        println!("number is non-zero");
    }
}

If Without Else

What happens if the if condition is false and there is no else block?

The if block is skipped and execution continues with the next statement.

fn main() {
    let number = 7;
    if number < 5 {
        println!("Less than 5");
    }
    println!("This always runs");
}

Else If Behavior

How does else if work in Rust?

It checks multiple conditions in sequence, executing only the first true branch.

fn main() {
    let number = 6;
    if number % 4 == 0 {
        println!("Divisible by 4");
    } else if number % 3 == 0 {
        println!("Divisible by 3");
    } else {
        println!("Other");
    }
}

If in Let Statement

Can if be used in a let statement?

Yes, but all branches must return the same type.

fn main() {
    let condition = true;
    let number = if condition { 5 } else { 6 };
    println!("number = {number}");
}

Three Loop Types

What are the three loop types in Rust?

loop, while, and for.

fn main() {
    loop { break; } // infinite unless broken
    while false {}  // runs while condition is true
    for _ in 0..1 {} // iterates over a range
}

Loop Behavior

How does loop work in Rust?

It repeats code indefinitely until explicitly stopped with break.

fn main() {
    loop {
        println!("again!");
        break;
    }
}

Loop Return Value

How can you return a value from a loop?

Use break followed by the value.

fn main() {
    let mut counter = 0;
    let result = loop {
        counter += 1;
        if counter == 3 {
            break counter * 2;
        }
    };
    println!("result = {result}");
}

Loop Labels

What are loop labels in Rust used for?

To specify which loop break or continue applies to in nested loops.

fn main() {
    'outer: loop {
        loop {
            break 'outer;
        }
    }
}

While Loop Behavior

How does a while loop work in Rust?

It runs as long as its condition evaluates to true.

fn main() {
    let mut n = 3;
    while n > 0 {
        println!("{n}");
        n -= 1;
    }
}

While Loop Errors

Why is looping through a collection with while potentially error-prone?

Manual index management can cause out-of-bounds errors or missed elements.

fn main() {
    let arr = [1, 2, 3];
    let mut i = 0;
    while i < arr.len() {
        println!("{}", arr[i]);
        i += 1;
    }
}

For Loop Preference

Why is a for loop preferred over a while loop for iterating collections?

It's safer, more concise, and avoids index errors.

fn main() {
    let arr = [1, 2, 3];
    for val in arr {
        println!("{val}");
    }
}

For Loop Countdown

How can you run a countdown using a for loop in Rust?

Use a reversed range.

fn main() {
    for number in (1..4).rev() {
        println!("{number}!");
    }
    println!("LIFTOFF!!!");
}

If Let Type Requirement

What is the key requirement when using if in a let statement?

All branches must evaluate to the same type.

fn main() {
    let condition = true;
    // let x = if condition { 5 } else { "six" }; // ❌ type mismatch
    let x = if condition { 5 } else { 6 }; // ✅ both i32
    println!("{x}");
}
