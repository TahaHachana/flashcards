What Are Compound Types

What are compound types in Rust?

Types that group multiple values together. Rust has two primitive compound types: tuples (group different types) and arrays (collection of same type). Both are fixed-size and stack-allocated.

Creating Tuples

How do you create a tuple in Rust?

Write comma-separated values inside parentheses. Type annotation is optional.

let tup: (i32, f64, u8) = (500, 6.4, 1);
let tup2 = (500, 6.4, 1);  // Type inferred

Accessing Tuple Elements

How do you access elements of a tuple?

Use dot notation with zero-based indices.

let tup = (500, 6.4, 1);
let first = tup.0;   // 500
let second = tup.1;  // 6.4
let third = tup.2;   // 1

Destructuring Tuples

How do you destructure a tuple into separate variables?

Use pattern matching with let binding.

let tup = (500, 6.4, 1);
let (x, y, z) = tup;
println!("{}, {}, {}", x, y, z);  // 500, 6.4, 1

Partial Tuple Destructuring

How do you ignore specific values when destructuring a tuple?

Use underscore (_) for values you don't need.

let tup = (500, 6.4, 1);
let (x, _, z) = tup;  // Ignore middle value
println!("{}, {}", x, z);  // 500, 1

The Unit Type

What is the unit type in Rust?

A tuple with no elements, written as (). Represents an empty value or empty return type. Functions that don't return a value implicitly return ().

let unit = ();
fn do_something() { }  // Returns ()

Returning Multiple Values with Tuples

How can functions return multiple values using tuples?

Return a tuple containing multiple values, then destructure it.

fn calculate(x: i32) -> (i32, i32) {
    (x * 2, x * 3)
}
let (doubled, tripled) = calculate(5);

Tuple Mutability

How do you modify tuple elements?

Make the tuple mutable, then assign to specific elements. Size and types cannot change.

let mut tup = (1, 2, 3);
tup.0 = 10;
println!("{:?}", tup);  // (10, 2, 3)

Tuple Limitations

What are the main limitations of tuples?

Fixed size (can't add/remove elements), no methods (like push/pop), limited practical length (~12 elements), and not directly iterable in for loops.

Creating Arrays

How do you create an array in Rust?

Use square brackets with comma-separated values of the same type.

let arr = [1, 2, 3, 4, 5];
let arr2: [i32; 5] = [1, 2, 3, 4, 5];

Array Type Annotation

What does the array type annotation [T; N] mean?

[T; N] means an array of N elements of type T. For example, [i32; 5] is an array of 5 i32 values.

let arr: [i32; 5] = [1, 2, 3, 4, 5];

Array with Same Values

How do you create an array with the same value repeated?

Use [value; count] syntax.

let arr = [3; 5];  // [3, 3, 3, 3, 3]
let zeros = [0; 100];  // 100 zeros

Accessing Array Elements

How do you access array elements?

Use square bracket indexing with zero-based indices of type usize.

let arr = [1, 2, 3, 4, 5];
let first = arr[0];   // 1
let last = arr[4];    // 5

Array Bounds Checking

What happens when you access an invalid array index?

Rust checks bounds at runtime and panics if the index is out of bounds, preventing buffer overflow vulnerabilities.

let arr = [1, 2, 3];
let x = arr[10];  // ❌ Panics: index out of bounds

Safe Array Access

How do you safely access an array element that might be out of bounds?

Use the get method which returns an Option instead of panicking.

let arr = [1, 2, 3];
match arr.get(10) {
    Some(val) => println!("Value: {}", val),
    None => println!("Out of bounds"),
}

Iterating Over Arrays

How do you iterate over an array?

Use a for loop directly, or use iter() with enumerate() for indices.

let arr = [10, 20, 30];
for element in arr {
    println!("{}", element);
}

for (i, val) in arr.iter().enumerate() {
    println!("arr[{}] = {}", i, val);
}

Array Length

How do you get the length of an array?

Use the len() method.

let arr = [1, 2, 3, 4, 5];
let length = arr.len();  // 5

Mutable Arrays

How do you modify array elements?

Make the array mutable, then assign to specific indices. Length cannot change.

let mut arr = [1, 2, 3, 4, 5];
arr[0] = 10;
arr[4] = 50;
println!("{:?}", arr);  // [10, 2, 3, 4, 50]

Array Slices

How do you create a slice of an array?

Use a reference with a range. Slices have type &[T] and provide a view into the array.

let arr = [1, 2, 3, 4, 5];
let slice = &arr[1..4];  // [2, 3, 4]

Multidimensional Arrays

How do you create and access multidimensional arrays?

Nest arrays and use multiple indices.

let matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]];
let element = matrix[1][2];  // 6
let grid: [[i32; 3]; 3] = matrix;

Arrays vs Tuples Element Types

What is the key difference between arrays and tuples regarding element types?

Arrays must have all elements of the same type. Tuples can have elements of different types.

let arr = [1, 2, 3];  // All i32
let tup = (1, 2.5, "hi");  // Different types

Arrays vs Tuples Access Method

How do access methods differ between arrays and tuples?

Arrays use bracket notation (arr[0]), tuples use dot notation (tup.0).

let arr = [1, 2, 3];
let first_arr = arr[0];

let tup = (1, 2, 3);
let first_tup = tup.0;

When to Use Arrays

When should you use arrays?

When you want stack allocation, know element count at compile time, size never changes, need fast indexed access, or memory layout matters (e.g., C interop). Examples: buffers, lookup tables, RGB values.

When to Use Tuples

When should you use tuples?

When grouping a small number of related values with different types, returning multiple values from functions, representing coordinates, or temporary grouping without creating a struct.

Arrays vs Vectors

What is the difference between arrays and vectors?

Arrays are fixed-size and stack-allocated. Vectors (Vec) are dynamic-size and heap-allocated. Use arrays when size is known and fixed, vectors when size needs to change.

let arr = [1, 2, 3];  // Fixed
let mut vec = vec![1, 2, 3];
vec.push(4);  // Can grow

Tuple Iteration

Can you iterate over tuples directly in a for loop?

No. Tuples are not directly iterable. Arrays can be iterated but tuples cannot.

let tup = (1, 2, 3);
for x in tup { }  // ❌ Error

let arr = [1, 2, 3];
for x in arr { }  // ✅ OK

Fixed Size Constraint

Can you change the size of arrays or tuples after creation?

No. Both arrays and tuples have fixed size determined at compile time. You cannot add or remove elements.

let mut arr = [1, 2, 3];
arr.push(4);  // ❌ Error: no push method

Stack Allocation

Where are arrays and tuples stored in memory?

Both are stored on the stack for fast access. Their fixed size at compile time enables stack allocation.

Nested Tuples

Can tuples contain other tuples?

Yes. Tuples can be nested. Access using chained dot notation.

let nested = ((1, 2), (3, 4));
let first_pair = nested.0;  // (1, 2)
let first_element = nested.0.0;  // 1

Array Bounds Safety

How does Rust prevent buffer overflow vulnerabilities with arrays?

Rust checks array bounds at runtime and panics if an index is out of bounds, unlike C/C++ which allows unsafe access.

let arr = [1, 2, 3];
let x = arr[5];  // ❌ Panics instead of accessing invalid memory
