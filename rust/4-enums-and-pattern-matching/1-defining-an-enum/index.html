Enum Definition Basics

How do you define a simple enum in Rust with two variants?

Use the enum keyword followed by variant names:

enum IpAddrKind {
    V4,
    V6,
}

Unit Tuple Struct Variants

What forms can enum variants take in Rust?

Variants can be:

  • Unit-like (no data): Quit
  • Tuple-like (unnamed values): Write(String)
  • Struct-like (named fields): Move { x: i32, y: i32 }

Enum With Data Example

Show an enum where each variant carries different kinds of data.

enum Message {
    Quit,                         // no data
    Move { x: i32, y: i32 },      // struct-like
    Write(String),                // tuple-like
    ChangeColor(i32, i32, i32),   // multiple values
}

Methods On Enums

How do you implement methods on an enum?

Use an impl block:

impl Message {
    fn call(&self) {
        println!("{:?}", self);
    }

    fn is_quit(&self) -> bool {
        matches!(self, Self::Quit)
    }
}

Deriving Traits For Enums

How can you automatically add traits like Debug or Clone to an enum?

Use the #[derive] attribute:

#[derive(Debug, Clone, PartialEq, Eq)]
enum Mode {
    Normal,
    Insert,
    Visual,
}

Default Enum Variant

How do you specify a default enum variant?

Mark a variant with #[default] and derive Default:

#[derive(Default)]
enum Mode {
    #[default]
    Normal,
    Insert,
    Visual,
}

Enum Discriminants

What are discriminants in enums and how do you set them?

Discriminants are explicit integer values for C-like enums:

enum Status {
    Ok = 0,
    NotFound = 1,
    Forbidden = 2,
}

They can be accessed with casts, e.g., Status::Ok as i32.

Enum Repr Attribute

What does #[repr(u8)] on an enum do?

It fixes the enum’s underlying representation for FFI or layout control:

#[repr(u8)]
enum Status {
    Ok = 0,
    NotFound = 1,
}

Nested Enum Data

How can enums contain complex types and be destructured?

struct Point { x: i32, y: i32 }

enum Shape {
    Circle(i32),
    Rectangle(Point, Point),
}

let s = Shape::Rectangle(Point { x: 0, y: 0 }, Point { x: 10, y: 5 });

match s {
    Shape::Circle(r) => println!("radius = {r}"),
    Shape::Rectangle(Point { x, .. }, Point { y, .. }) =>
        println!("corner at ({x}, {y})"),
}

Option Enum Usage

What does the Option<T> enum represent?

Option<T> represents either:

  • Some(T) — a present value.
  • None — absence of a value.
let x: Option<i32> = Some(5);
let y: Option<i32> = None;

Result Enum Usage

What does the Result<T, E> enum represent?

Result<T, E> represents:

  • Ok(T) for success.
  • Err(E) for failure.
fn divide(a: i32, b: i32) -> Result<i32, String> {
    if b == 0 {
        Err("division by zero".to_string())
    } else {
        Ok(a / b)
    }
}
