## git log Default Output Fields

What four pieces of information does a standard `git log` entry display for each commit?

---

```
1. Full SHA-1 hash   40-character unique commit identifier
                     (e.g., 521edbe1dd2ec9c6f959c504d12615a...)

2. Author            Name and email of the person who wrote
                     the change.

3. Date              The author timestamp (when the work was
                     done), including timezone.

4. Commit message    The full log message, starting with the
                     subject line (indented in output).
```

Commits are listed in **reverse chronological order** — newest first.
