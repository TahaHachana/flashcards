## The Three States of Git Files

What are the three states a file can be in within Git, and where does each reside?

---

```
State       | Location              | Meaning
------------|----------------------|----------------------------------------
Modified    | Working Directory    | Changed but not yet committed
Staged      | Staging Area (Index) | Marked to go into the next commit
Committed   | .git Repository      | Safely stored in the local database
```
