## Git vs. GitHub — Core Distinction

What is the fundamental difference between Git and GitHub?

---

**Git** is a free, open-source, distributed version control tool that runs on your local machine.

**GitHub** is a cloud-based platform (owned by Microsoft) that *hosts* Git repositories and adds collaboration features on top — such as pull requests, issues, and CI/CD pipelines.

The analogy: Git is to GitHub as email is to Gmail.
