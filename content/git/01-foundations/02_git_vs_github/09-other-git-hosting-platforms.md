## Other Git Hosting Platforms

Name three Git hosting platforms other than GitHub and GitLab, and a distinguishing trait of each.

---

```
Platform        | Notable For
----------------|------------------------------------------
Bitbucket       | Atlassian ecosystem (Jira, Confluence)
Gitea / Forgejo | Lightweight self-hosted; minimal resources
AWS CodeCommit  | Managed Git hosting within the AWS ecosystem
```

There are many more (Azure DevOps Repos, Sourcehut, etc.). GitHub and GitLab are simply the most widely used.
