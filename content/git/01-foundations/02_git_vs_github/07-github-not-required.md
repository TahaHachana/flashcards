## Is GitHub Required to Use Git?

Do you need GitHub (or any platform) to use Git?

---

**No.** Git is completely independent of any platform. You can:
- Use Git entirely on your local machine with no internet connection
- Host a bare Git repo on any server you control
- Use Git across a team via SSH without any third-party service

GitHub, GitLab, and others are optional conveniences that add web-based collaboration on top of Git.
