## Pull Requests Are a Platform Feature

Are pull requests a Git feature? Explain.

---

**No.** Pull requests (or merge requests on GitLab) are a **platform feature** — they exist in GitHub, GitLab, and similar services, but not in Git itself.

Git's built-in mechanisms are `git merge`, `git fetch`, and `git pull`. The concept of a "pull request" as a formalized, reviewable, discussable code change proposal was introduced by hosting platforms to facilitate team collaboration.
