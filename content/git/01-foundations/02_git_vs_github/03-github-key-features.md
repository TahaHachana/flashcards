## GitHub Key Features

What are five key features that GitHub adds on top of Git?

---

1. **Pull Requests (PRs)** — formal code review and merge proposals
2. **Issues** — built-in bug tracker and feature request system
3. **GitHub Actions** — CI/CD automation platform
4. **GitHub Pages** — free static website hosting from a repo
5. **Organizations & Teams** — access control and team management

GitHub is owned by Microsoft and available at github.com.
