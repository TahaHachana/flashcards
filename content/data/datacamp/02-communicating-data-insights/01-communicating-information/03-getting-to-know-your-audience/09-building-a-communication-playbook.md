## Building a Communication Playbook

What is a communication playbook in this context, and what triggers the need to create one?

---

A **communication playbook** is a tailored strategy for each stakeholder interaction, built by answering the three key audience questions (role, knowledge, familiarity).

It is created when a communicator needs to deliver insights to **multiple stakeholders** with different roles, technical levels, and prior relationships — ensuring each interaction is appropriately targeted rather than using a one-size-fits-all approach.
