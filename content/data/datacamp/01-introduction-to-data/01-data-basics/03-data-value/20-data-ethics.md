## Data Ethics

What is data ethics? Name its four core principles and link each to an example from this lesson.

---

Data ethics is the set of moral principles guiding the responsible collection, storage, and use of data.

| Principle | Lesson Example |
|---|---|
| **Consent** | Wearable data is only shared with physicians if the individual agrees. |
| **Transparency** | Organizations openly share ROI data internally to guide decisions. |
| **Fairness** | Student data should be used to help all learners, not disadvantage any group. |
| **Consideration of outcomes** | Patient data sharing must weigh individual privacy against research benefit. |
