## Data Life Cycle Cloze

Complete the data life cycle framework:

A ___ question is defined → data is ___ from sources → stored in ___ → cleaned to remove ___ → analyzed to extract ___ → ___ with stakeholders → finally ___ or ___.

---

A **business** question is defined → data is **collected** from sources → stored in **databases or data warehouses** → cleaned to remove **errors and inconsistencies** → analyzed to extract **insights** → **shared** with stakeholders → finally **archived** or **destroyed**.
