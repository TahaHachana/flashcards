## Common Aggregation Types

Name five common aggregation types and give a brief description of each.

---

1. **Average (mean)** — the central value of a dataset; the most commonly encountered aggregation in daily life.
2. **Sum (total)** — the combined value of all data points.
3. **Minimum** — the smallest value in a dataset.
4. **Maximum** — the largest value in a dataset.
5. **Mode** — the most frequently occurring value in a dataset.
