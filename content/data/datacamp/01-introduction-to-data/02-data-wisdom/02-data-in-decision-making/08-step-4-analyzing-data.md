## Step 4 — Analyzing Data

What is the purpose of the analysis step, and what are six common tools used for it?

---

Analysis transforms data into something actionable — insights that can inform a decision. Common data analysis tools include:
1. Python
2. R
3. Tableau
4. Power BI
5. Excel
6. Google Sheets

Each tool supports different types of analysis; the right choice depends on the complexity of the task and the analyst's skillset.
