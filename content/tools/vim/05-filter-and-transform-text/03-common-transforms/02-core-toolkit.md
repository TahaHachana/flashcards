## The Core Transform Toolkit

Name the common Unix commands worth knowing as Vim filters and what each does.

---

```
sort     order lines (alpha or numeric)
uniq     collapse adjacent duplicate lines
fmt/par  reflow/wrap prose to a width
column   align fields into columns
tr       translate/squeeze/delete characters
sed      stream editor (substitutions, deletions)
awk      field-aware processing and reshaping
```
