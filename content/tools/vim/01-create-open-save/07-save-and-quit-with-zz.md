## Save and Quit with ZZ

What does the `ZZ` command do and what is important to remember about it?

---

`ZZ` (capital Z twice) writes (saves) the buffer to the file and then quits Vim, returning you to the shell prompt.

Key points:
- Must be in command mode first (press ESC if unsure).
- It is case-sensitive — both letters must be uppercase.
- It is equivalent to `:wq`.
