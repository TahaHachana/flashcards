## Register Summary Table

Summarize the register types and how to put from each.

---

```
Register        Holds                                  Put with
--------------  -------------------------------------  --------
" (unnamed)     last delete/yank (often overwritten)   p / P
1-9 (numbered)  last nine line-sized deletions         "Np
a-z (named)     text you explicitly yank/delete in     "ap / "aP
A-Z (named)     appends to the matching a-z register   "Ap
```
