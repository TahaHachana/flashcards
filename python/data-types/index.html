Dictionaries

Dictionaries are collections of key-value pairs. They are unordered, changeable and indexed.

person = {"name": "Bob", "age": 30}

Sets

Sets are unordered mutable collections of unique elements.

unique_numbers = {1, 2, 3}

Strings

A string in Python is a sequence of letters in a specific order.

double_quotes = "Hello World"
single_quotes = 'Hello World'
triple_quotes = """Hello World"""

Lists

Lists are a collection of items that are ordered and changeable. They are defined by square brackets [] and items are separated by commas. Lists can contain items of different types.

fruits = ["apple", "banana", "cherry"]

Tuples

A tuple is a sequence of ordered and immutable objects.

coordinates = (10.0, 20.0)

Numbers

Python has three types of numbers:

  • int
  • float
  • complex
x = 1    # int
y = 2.8  # float
z = 1j   # complex
