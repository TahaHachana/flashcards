Window Command Prefix

Cloze: Every Vim window command begins with the prefix {{CTRL-W}}, where W is mnemonic for "window".

CTRL-W

Horizontal Vs Vertical Split

Which commands create a horizontal split and a vertical split?

:split creates a horizontal split (windows stacked top/bottom); :vsplit creates a vertical split (windows side by side). Both default to showing the same file unless a filename is given.

Full Split Command Syntax

What are the four optional parts of :[n]split [++opt] [+cmd] [file]?

n = height in lines of the new window; ++opt = an option passed to the new window; +cmd = a command to run in the new window; file = the file to open in it.

Split With Options Example

Write a command to open a 15-line-tall split editing otherfile with Unix file format.

:15split ++fileformat=unix otherfile

Split Keystroke Shortcuts

What are the keystroke shortcuts for a horizontal split and a vertical split?

CTRL-W s (or CTRL-W CTRL-S) splits horizontally; CTRL-W v (or CTRL-W CTRL-V) splits vertically.

Window Navigation Directions

How do you move to the window below, above, left, and right of the current one?

CTRL-W j   window below
CTRL-W k   window above
CTRL-W h   window to the left
CTRL-W l   window to the right

These mirror the h/j/k/l motion keys.

Cycle Through Windows

Which command cycles through all windows (wrapping around), unlike the directional moves?

CTRL-W w (or CTRL-W CTRL-W) moves to the next window and cycles through all of them, restarting at the top-left when it reaches the end. CTRL-W W goes the opposite direction.

Count With Window Navigation

How would you jump three windows down from the current window?

3 CTRL-W j — prefix the window motion with a count.

Enable Mouse In Terminal Vim

How do you enable mouse window navigation in terminal Vim?

:set mouse=a, which activates the mouse for command line, input, and navigation. (gvim supports the mouse by default.)

Relayout Window Commands

What do CTRL-W H/J/K/L do to the current window?

They move the current window to a screen edge and resize it to fill that edge: K = top (full width), J = bottom (full width), H = left (full height), L = right (full height).

Move Window To New Tab

Which keystroke moves the current window into its own new tab?

CTRL-W T. (No action is taken if it is the only window in the current tab.)

Resize Window Height

Which keystrokes increase, decrease, and equalize window height?

CTRL-W + increases height, CTRL-W - decreases height (default one line each), and CTRL-W = makes all windows equal size.

Resize Window Width

Which keystrokes decrease and increase window width, and what's the mnemonic?

CTRL-W < decreases width and CTRL-W > increases width. Mnemonic: << shift left (narrower), >> shift right (wider). CTRL-W | maximizes width.

Absolute Resize Commands

How do you set an absolute window height and width with ex commands?

:resize n sets height to n lines; :vertical resize n sets width to n columns. (:resize +n / -n make relative changes.)

Laststatus Option

What do the values of the laststatus option mean?

0 = never show a status line; 1 = show one only when there are multiple windows (default); 2 = always show a status line for every window. Best set in .vimrc.

Equalalways Option

What does the equalalways option do, and which option controls its direction?

equalalways makes Vim resize all windows equally whenever you split or close one. eadirection (values hor, ver, both) controls which directions are equalized.

Close Vs Quit Window

What are the commands/keystrokes to quit the current window, close it, and make it the only window?

:quit / CTRL-W q quits the window; :close / CTRL-W c closes it; :only / CTRL-W o makes it the only window. None abandon unsaved changes without !.

No Ctrl-W Ctrl-C

Why is there no CTRL-W CTRL-C close-window keystroke?

Because Vim uses CTRL-C to cancel commands, so CTRL-W CTRL-C just cancels the command. Use CTRL-W c (with a lowercase c, no Ctrl) instead.

Creating And Closing Tabs

Which commands open a new tab, close the current tab, and close all other tabs?

:tabnew [file] (or :tabedit [file]) opens a new tab; :tabclose closes the current tab; :tabonly closes all other tabs.

Moving Between Tabs

How do you move one tab right and one tab left from the keyboard?

CTRL-PageDown moves one tab to the right and CTRL-PageUp moves one tab to the left, wrapping around at the ends. (Also :tabnext / :tabprevious.)

Open Files In Separate Tabs At Launch

Which command-line option opens each file in its own tab at startup?

The -p option, e.g. vim -p ~/.vimrc /etc/hosts opens each file in a separate tab.
