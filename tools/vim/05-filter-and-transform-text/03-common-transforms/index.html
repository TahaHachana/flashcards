Why Vim Needs No Built-In Transforms

Why can Vim sort, dedupe, reflow, and reshape text without having any of these features built in?

Because the filter operator pipes lines to any external Unix command and swaps in its output. The editor doesn't need to know the tools — it just delegates to them.

The Core Transform Toolkit

Name the common Unix commands worth knowing as Vim filters and what each does.

sort     order lines (alpha or numeric)
uniq     collapse adjacent duplicate lines
fmt/par  reflow/wrap prose to a width
column   align fields into columns
tr       translate/squeeze/delete characters
sed      stream editor (substitutions, deletions)
awk      field-aware processing and reshaping

Numeric Versus Alphabetic Sort

What is the difference between sort and sort -n?

sort orders lines lexicographically (so "100" comes before "9"); sort -n orders them numerically (so 9 comes before 100).

Sort And Dedupe In One Step

Cloze: To sort a block AND remove duplicates in a single filter, use {{!sort -u}}.

!sort -u — the -u flag sorts and removes duplicate lines together.

Why uniq Usually Follows sort

Why is uniq almost always preceded by sort?

uniq only removes ADJACENT duplicate lines. Unsorted input leaves non-neighboring duplicates intact, so you sort first to bring duplicates together.

Reflow A Paragraph To A Width

What filter reflows the current paragraph to 72 columns?

!}fmt -w 72

Align Fields Into Columns

Which command, used as a filter, pads whitespace-separated fields so they line up into a table?

column -t (e.g. !}column -t).

tr Operates On What Unit

Does tr work line by line or character by character, and what does tr '[:lower:]' '[:upper:]' do?

tr works character by character. tr '[:lower:]' '[:upper:]' converts lowercase letters to uppercase across the filtered text.

sed Substitution Flags

What is the difference between sed 's/old/new/' and sed 's/old/new/g'?

The first replaces only the FIRST match on each line; the g flag in the second replaces ALL matches on each line.

awk Fields And NF

In awk, what do $1, $2, and NF refer to?

$1, $2, ... are the first, second, etc. fields of the current line; NF is the number of fields on that line.

Swap Two Fields With awk

What awk program, used as a filter, swaps the first two fields of every line?

awk '{print $2, $1}'

Setting awk's Field Separator

How do you make awk split on punctuation as well as spaces, e.g. so "car", "car," and "car." count as one word?

Set the field separator to a regex in a BEGIN rule: BEGIN { FS = "[,. ]+" }.

Pipes Amplify Filters

How do pipes increase the power of Vim filters?

You can chain multiple Unix commands in one filter, e.g. :%!awk '...' | sort -n, so the buffer is transformed by the whole pipeline at once.

The Authors' Favorite Filter

Which command do the book's authors reach for most often when filtering Vim buffers, and what others do they routinely use?

They favor awk, and routinely also use sed, grep, wc, head, tail, and sort, among others.
