Basic Substitute Syntax

What does :s/old/new/ do?

It changes the first occurrence of old to new on the current line only.

The Substitution Delimiter

In :s/old/new/, what is the role of /, and when can the trailing one be omitted?

The / is the delimiter separating the parts of the command. The trailing delimiter is optional when it is the last character on the line (so :s/old/new works). Almost any punctuation character can serve as the delimiter.

The g Flag

What does the g flag do in :s/old/new/g?

It changes every occurrence of old to new on the line, not just the first occurrence.

g Flag Versus g Command

What is the difference between the g flag in :s///g and the :g command?

The g flag affects each matching pattern on a single line. The separate :g command selects each line of the file that matches a pattern. They are different mechanisms that share the letter g.

Substitute With A Line Range

How do you change every occurrence of old to new from line 50 to line 100?

:50,100s/old/new/g

Substitute Across The Whole File

Give two equivalent commands that change every occurrence of old to new in the entire file.

:1,$s/old/new/g and :%s/old/new/g — % is shorthand for 1,$ (first line through last line).

Meaning Of The Percent Sign

What range does % represent as a substitute address?

All lines in the file (equivalent to 1,$).

The c Confirm Flag

What does adding c to a substitute command do, e.g. :1,30s/his/the/gc?

It prompts for confirmation before each replacement. ex highlights the match and asks replace with ... (y/n/a/q/l/^E/^Y)? for each one.

Confirmation Prompt Responses

What do the substitution-confirmation responses y, n, a, q, and l mean?

y   substitute this match
n   skip this match
a   substitute this and all remaining matches
q   quit substituting
l   substitute this match and then quit ("last")

(CTRL-E / CTRL-Y scroll the screen; ESC quits substituting.)

Line Addressing Symbols

List the common ex line-addressing symbols you can use as a substitute range.

n          Line number n
.          Current line
$          Last line
%          All lines in the file
.+n        Current line plus n
.-n        Current line minus n
/pattern/  Forward search for first matching line
?pattern?  Backward search for first matching line

Undoing A Substitution

Can u undo a substitution, and what is the limitation?

Yes, u undoes a substitution, but only if it was your most recent edit. Because you may not catch a bad change in time, :w before and :e! to revert are safer protections.

Spot Editing Loop Alternative

When you want to change only some matches (not all), what vi loop lets you decide case by case?

The n (repeat search) and . (repeat last change) loop. For example: /which then cwthat ESC, then press n to find the next match and . to apply the same change only where appropriate.

Where To Find All Substitute Flags

Which help command lists all the available substitute flags in Vim?

:help s_flags
