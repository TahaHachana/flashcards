What Is The Relationship Between vi And ex

What is vi (and Vim) in relation to ex?

vi is the visual (screen) mode of the underlying line editor ex. The keystroke commands are a front end; ex is where the power for large-scale, file-wide editing lives, accessed through the colon (:) prompt.

The Lineage Of ex

ex is descended from which earlier Unix editors?

ex descends from the Unix line editor ed, which was itself based on an earlier line editor called QED. This line-editor heritage is why ex is organized around line numbers and addresses.

Why Line Editors Used Line Numbers

Why did line editors like ex evolve around line numbers?

Before screen editors, people worked on printing terminals — printing a line, issuing a command to change it, then reprinting to check. Line numbers were the way to quickly identify which part of a file to edit.

When ex Beats vi

When does ex's line orientation give it an advantage over vi's normal-mode commands?

When making large-scale changes that affect many parts of a file (or multiple files at once). vi is simpler for single small edits, but ex can modify large blocks of text with a single command.

Invoke ex From The Shell

How do you start the ex line editor directly from the shell, and what do you see?

Run ex filename. You get a line count and a : prompt (e.g. "practice" 8L, 261B). No file lines are shown until you issue a command that displays them.

Enter ex From Within vi

Which vi command drops you into ex command mode?

Q. (Typing Q accidentally is a common way people end up at an unexpected ex colon prompt.)

Return To Visual Mode From ex

How do you return to the visual (vi) editor when you are in ex?

Type :vi (or just vi at the ex prompt) and press ENTER.

Syntax Of An ex Command

What is the general syntax of an ex command issued from vi?

:[address] command [options] — the : signals ex, the address is the line or range to act on, then the command and any options. Press ENTER to execute.

Default Address When None Given

If you give an ex command with no line address, which line does it affect?

The current line.

The Basic Print Command

What does p do in ex, and what is a bare line number equivalent to?

p prints (displays) lines to the screen. A bare line number is equivalent to a print command for that line, so :1 and :1p both display line 1.

Spaces As Separators In ex

Where can you use spaces to make an ex command more readable, and where can't you?

You can separate addresses, patterns, and commands with spaces. You cannot use a space as a separator inside a pattern or at the end of a substitute command.

Vim As ex

How does Vim relate to ex?

Vim provides the underlying ex editor with many enhancements over the original. On systems where vi is Vim, ex usually also invokes Vim in ex mode.
