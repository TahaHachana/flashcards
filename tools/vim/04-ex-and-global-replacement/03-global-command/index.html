The Global Command Form

What is the form of the global command, and what does it do?

:g/pattern/command — ex scans the entire buffer, remembers every line matching pattern, then executes command on each of those lines.

g Selects Lines Not Matches

What does the :g command select, in contrast to what :s operates on?

:g selects which lines to act on (the lines matching a pattern). :s operates within a line. So :g chooses lines and :s changes text inside them.

Delete All Lines Matching A Pattern

How do you delete every line containing a "# FIXME" comment?

:g/# FIXME/d

Other Commands g Can Drive

Besides :s, which ex commands can :g be combined with?

:d (delete), :m (move), :co (copy), and others — letting you make global deletions, moves, and copies, not just substitutions.

Restrict g To A Line Range

How do you display every line containing pattern, but only between lines 60 and 124?

:60,124g/pattern/p

Inverting The Match

How do you run a command on lines that do NOT contain a pattern?

Use :g! or its synonym :v. For example, :v/#include/d deletes every line that is not an "#include" line.

v Is Equivalent To g Bang

What is :v/pattern/command equivalent to?

:g!/pattern/command — it executes the command on all lines that do not match the pattern.

Context Sensitive Replacement Syntax

What is the syntax for a context-sensitive replacement using :g and :s?

:g/pattern/s/old/new/g — operate only on lines matching pattern, and on those lines substitute old with new (the trailing g replaces all matches per line).

The Three g Meanings

In :g/pattern/s/old/new/g, what do the leading :g and the trailing g each control?

The leading :g chooses which lines to act on (lines matching pattern). The trailing g flag makes the substitution apply to every match on each chosen line, not just the first.

Context Sensitive Replacement Example

Write a command that changes Esc to ESC, but only on lines containing class="keycap".

:g/class="keycap"/s/Esc/ESC/g

Reusing The Find Pattern

When the line-finding pattern is the same string you want to change, how can you avoid typing it twice?

Leave the substitute pattern empty. :g/string/s//new/g finds lines containing string and replaces that same string with new.

g Empty Substitute Equals Percent s

What plain substitute command is equivalent to :g/editer/s//editor/g?

:%s/editer/editor/g. They have the same effect; the :%s form is shorter when the find-pattern and the change-pattern are identical.
