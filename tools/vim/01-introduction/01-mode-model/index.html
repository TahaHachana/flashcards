vi/Vim Is A Moded Editor

What does it mean that vi/Vim is a "moded" editor?

The same key does different things depending on the current mode. In insert mode keystrokes become text; in command mode each key is an instruction. You can picture each mode as a separate keyboard.

Mode On File Open

When you first open a file in Vim, which mode are you in?

Command (normal) mode — the editor is waiting for a command, not for text. This is why typing immediately does not just enter text.

Command Mode Purpose

What is command (normal) mode used for?

Issuing commands: moving the cursor, editing (change/delete/copy), entering insert mode, and quitting/saving the file. Keystrokes are interpreted as instructions, not text.

Insert Mode Purpose

What is insert mode used for?

Adding new text. In insert mode the keyboard behaves normally and everything you type becomes content in the buffer.

Three Properties Of Command-Mode Commands

Name the three defining properties of vi/Vim command-mode commands.

They are case sensitive (i differs from I), not echoed on screen as you type them, and do not require pressing ENTER to execute.

The Third Mode

What is often called Vim's "third mode"?

The colon / ex command line — the bottom-line prompt where you type ex commands (and searches). Its commands are echoed and need ENTER to run.

Bottom-Line Command Symbols

Which symbols start a bottom-line (echoed) command, and what does each begin?

:   an ex command          (e.g. :wq)
/   a forward search       (e.g. /pattern)
?   a backward search      (e.g. ?pattern)

vi Is The Visual Mode Of ex

What is the relationship between vi and the ex editor?

ex is the underlying line editor, and vi is really just its "visual" mode. The ex editor is always available from within vi via the colon (:) prompt.

The ESC Safety Habit

What should you do whenever Vim does not respond the way you expect, or you are unsure which mode you're in?

Press ESC once or twice. It forces you into command mode; if you were already there, the editor beeps — confirming you are safely in command mode.

Beep Mode

Why is command mode sometimes nicknamed "beep mode"?

Because pressing ESC while already in command mode makes the editor beep. The beep is the audible signal that you are in command mode.

Showmode Indicator

How can Vim visually tell you that you are in insert mode?

With the showmode option on, it displays an indicator such as "-- INSERT --" on the status line. Don't rely on it everywhere; the ESC-and-listen habit always works.
