Enter Insert Mode

From command mode, which key enters insert mode, and what does the letter stand for?

Press i — it stands for "insert." Text you then type is inserted before the character the cursor was on.

The i Command Is Not Echoed

When you press i to enter insert mode, does the i appear on screen?

No. The i is a command, not text, so it does not appear. Only what you type afterward is shown and added to the buffer.

Keystrokes iintroduction

On a fresh file, what appears on screen if you type the keystrokes "iintroduction"?

Just "introduction". The first i is read as the insert command; the rest is entered as text.

Leave Insert Mode

Which key leaves insert mode and returns you to command mode?

ESC.

Cursor Behavior On ESC

What happens to the cursor when you press ESC to leave insert mode?

It moves back one space so it rests on the last character you typed, then you are in command mode.

When Text Stops Being Text

After you give the insert command, how long are your keystrokes treated as text?

Every keystroke is treated as text until you press ESC.

Fixing Typos While Inserting

How do you correct a mistake while still in insert mode?

Backspace over the error and retype. Whatever you back over is deleted (whether or not the terminal visibly erases it).

Backspace Limit In vi-Compatible Mode

In strict vi-compatible mode, how far back can you backspace while inserting?

Only up to the point where you entered insert mode — you cannot backspace beyond it. With vi compatibility disabled (nocompatible, the common Linux default) you can backspace past that point.

Breaking Lines In Insert Mode

How do you start a new line while inserting text, by default?

Press ENTER to insert a line break. (The wrapmargin option can auto-break lines at a right margin once configured.)
