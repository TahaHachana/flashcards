The Three Visual Modes

What do v, V, and CTRL-V each select?

v = charwise (one character at a time), V = linewise (whole lines), CTRL-V = blockwise (a rectangular column block).

Order Of Operations

How does visual mode reverse the normal Vim editing order?

Normally you give the operator first, then the motion (d3w). In visual mode you select the text first (watching it highlight), then apply the operator to the selection.

Extending A Selection

Once in visual mode, how do you grow the highlighted region?

Use any motion command. Each motion (e.g. w, }, /pattern, j) moves the cursor and extends the highlight to the new position.

Operators On A Selection

After selecting text in visual mode, what do d, y, c, >, and < do?

d deletes the selection, y yanks (copies) it, c changes it, > indents it one shiftwidth right, < indents it one shiftwidth left.

Copy A Line Visually (Cloze)

Cloze: To copy whole lines without the mouse: press _____ to select linewise, extend the selection, then press y to yank.

V

Selecting A Whole Word

In visual mode, what does aw select, and how does it differ from iw?

aw selects "a word" including surrounding whitespace; iw selects the "inner word" without the whitespace. (aw/iw are punctuation-delimited; aW/iW are whitespace-delimited.)

Text-Object Selectors

Which visual-mode selectors grab a sentence or a paragraph around the cursor?

as/is for a sentence / inner sentence, and ap/ip for a paragraph / inner paragraph.

Blockwise Use Case

What can CTRL-V blockwise visual mode do that plain operator+motion cannot?

Select and operate on a rectangular column of text spanning multiple lines — e.g. delete or insert into the same columns across many lines.

Visual Mode Origin

Is visual mode available in the original vi?

No. Visual mode is a Vim-only feature; original vi has no visual mode.

Leaving Visual Mode

How do you exit visual mode without operating on the selection?

Press ESC, or press the same key you entered with (v, V, or CTRL-V) to toggle it off.
