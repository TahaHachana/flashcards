What A Mark Is

What is a mark in Vim?

An invisible bookmark at a position in the file. You set it with a letter name, edit elsewhere, then jump back. Marks exist only for the current session and are not stored in the file.

Setting A Mark

How do you set a mark, and what naming rule differs between vi and Vim?

m{x} sets mark x at the cursor (x is any letter). Original vi allows only lowercase letters; Vim distinguishes uppercase from lowercase.

Backquote Versus Apostrophe

What is the difference between `x and 'x?

`x (backquote) jumps to the exact character of mark x; 'x (apostrophe) jumps to the first non-blank character of the line holding mark x. Backtick = precise, apostrophe = line-level.

Bounce Back

What does `` (two backquotes) do?

It returns the cursor to the exact position it was at before the most recent jump — useful for peeking elsewhere and snapping back.

Double Apostrophe

What does '' (two apostrophes) do, compared with ``?

'' returns to the start of the line of the previous position (line-level), whereas `` returns to the exact character position.

Mark As Edit Target

What does da` do?

Deletes from the cursor up to and including the position of mark a. A mark can serve as the motion target of an operator.

Delete A Big Block With Marks

How can marks help delete a large block without counting lines?

Jump to one end and set a mark (e.g. ma), move to the other end, then dato delete the whole range — no need to compute a line count like47dd`.

Marks Are Session-Only (Cloze)

Cloze: Place markers are set only during the current session; they are _____ in the file.

not stored

Vim Automatic Marks

What do the Vim marks `. and '0 jump to?

`. jumps to the position of the last change in the file; '0 goes to where you were when you last exited Vim.

Listing Marks

How do you list all active marks in Vim?

:marks

Marks Quick Reference

Summarize the core mark commands.

Command   Effect
--------  -----------------------------------------------
m{x}      Set mark x at the cursor
`x        Jump to the exact character of mark x
'x        Jump to the start of the line holding mark x
``        Return to exact position before the last jump
''        Return to the line of that previous position
d`a       Delete from cursor up to mark a
:marks    List active marks (Vim)
