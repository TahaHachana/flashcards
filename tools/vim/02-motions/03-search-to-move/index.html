Search Forward

How do you search forward for a pattern?

/pattern then ENTER — searches forward from the cursor.

Search Backward

How do you search backward for a pattern?

?pattern then ENTER — searches backward from the cursor.

Where Search Begins And Wraps

Where does a search start, and what happens at the end of the file?

It starts at the cursor, moves in the search direction, and wraps around the end (or start) of the file by default, landing on the first occurrence.

Pattern Not Found

What message appears if a search finds no match?

"Pattern not found" (the exact wording varies by editor version).

Substring Matching

If you search for red, will it match only the whole word "red"?

No — a pattern is just a character string, so red matches red and also the red inside occurred.

Spaces In A Pattern

What happens if you include a space before or after the search pattern?

The space is treated as part of the text to be matched.

Repeat Search Same Direction

Which command repeats the last search in the same direction?

n — repeats the last search in the same direction.

Repeat Search Opposite Direction

Which command repeats the last search in the opposite direction?

N — repeats the last search in the opposite direction.

Repeat With Slash Or Question

Besides n/N, how can you repeat the last search forward or backward?

Press / then ENTER to repeat forward, or ? then ENTER to repeat backward (reusing the last pattern).

Disable Search Wrapping

Which option stops searches from wrapping around the file, and how do you turn it off?

wrapscan controls wrapping; :set nowrapscan disables it so searches stop at the bottom/top instead of wrapping.

Delete Through A Search

What does d/while followed by ENTER do?

It deletes from the cursor up to and through the next occurrence of while. The deletion is character-based, not whole lines.

Search As A Motion Target

How can a search act as a motion target for an operator?

Combine an operator with a search, e.g. d/pattern, c/pattern, or y/pattern (and the ? backward forms), to act on text from the cursor up to the match.

Character Versus Line Deletion

How does deleting with d/pattern differ from the ex command :.,/pattern/d?

d/pattern deletes character-by-character, only the text between the cursor and the match. :.,/pattern/d deletes the entire range of lines.

Search Offset

What does /pattern/+3 do?

It searches forward for pattern and lands 3 lines after the matched line (an offset). /pattern/-3 lands 3 lines before.

Word Under Cursor Search

In Vim, which keys search forward and backward for the whole word under the cursor?

* searches forward and # searches backward for the exact word under the cursor (g*/g# match it even inside longer words).

Search History In Vim

How does Vim help you reuse previous searches?

Vim keeps a search history; after pressing / or ? you can scroll through past searches with the up/down arrow keys.

Spot Edit Loop

Describe the search-then-. loop for fixing scattered occurrences of a word.

/which to find it, cwthat ESC to change it, n to jump to the next, then . to repeat the change (or n to skip). Decide case by case instead of a global substitution.
