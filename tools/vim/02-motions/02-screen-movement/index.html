Scroll Forward One Screen

Which command scrolls forward one full screen?

CTRL-F — scroll forward (page down) one full screen.

Scroll Backward One Screen

Which command scrolls backward one full screen?

CTRL-B — scroll backward (page up) one full screen.

Half Screen Scrolling

Which commands scroll a half screen down and a half screen up?

CTRL-D scrolls forward (down) a half screen; CTRL-U scrolls backward (up) a half screen.

Scroll One Line

What do CTRL-E and CTRL-Y do, and what is special about the cursor?

CTRL-E scrolls the window down one line (shows one more line at the bottom); CTRL-Y scrolls up one line (shows one more at the top). They do not move the cursor to the start of the line — it stays at the same point in its line.

Screen As A Window

What is the mental model for the screen relative to a file?

The file is like a long roll of paper; the screen is a window of about 24 lines onto that roll. Scrolling slides the window.

Reposition Current Line To Top

How do you move the current line to the top of the screen (give the classic and the Vim form)?

Classic: z ENTER (or z+). Vim: zt.

Center The Screen On The Cursor

Which command centers the current line in the middle of the screen (classic and Vim)?

Classic: z.. Vim: zz.

Reposition Current Line To Bottom

How do you move the current line to the bottom of the screen (classic and Vim)?

Classic: z-. Vim: zb.

Numeric Prefix With z

What does a number before a z command mean, e.g. 200z ENTER?

It is a line number, not a multiplier: 200z ENTER puts line 200 at the top of the screen.

The scrolloff Option

Why might z ENTER place the current line a few lines below the top instead of exactly at the top?

Because scrolloff is set to a nonzero value (often 5), keeping that many context lines above/below the cursor. Set :set scrolloff=0 to disable it.

Redraw The Screen

Which command redraws the screen without scrolling, and what is the Vim caveat?

CTRL-L redraws without scrolling. Caveat: CTRL-R redraws in classic vi but means redo in Vim, so use CTRL-L in Vim.

Home Middle Last Lines

What do H, M, and L do?

H moves to the top (home) line of the screen, M to the middle line, and L to the last (bottom) line — landing on the first nonblank character.

HML Mnemonic

What is the mnemonic for H, M, and L?

High, Middle, Low — top, middle, and bottom of the visible screen.

Count With H And L

What do 2H and 3L do?

2H moves to the line 2 lines below the top line; 3L moves to the line 3 lines above the bottom line.

Navigation Rhythm

What is a common three-step rhythm for getting around a long file with screen commands?

Scroll to the region with CTRL-F/CTRL-B (or CTRL-D/CTRL-U), land on a visible line with H/M/L, then recenter with zz to see context.
