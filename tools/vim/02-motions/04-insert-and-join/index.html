Insert Versus Append

What is the difference between i and a?

i inserts text before the cursor; a appends text after the cursor (the cursor moves one space right before you type).

Append At End Of Line

Which command appends text at the end of the current line, and what does it save?

A — appends at the end of the line. It saves a keystroke over $a.

Insert At Beginning Of Line

Which command inserts at the beginning (first nonblank) of the current line?

I — inserts at the beginning of the line, saving you from pressing ^ first.

Open Line Below

Which command opens a new empty line below the cursor and enters insert mode?

o (lowercase) — opens a new line below the cursor.

Open Line Above

Which command opens a new empty line above the cursor and enters insert mode?

O (uppercase) — opens a new line above the cursor.

Leave Insert Mode

After using any insert-mode entry command, how do you return to command mode?

Press ESC — every insert-class command (i a I A o O) leaves you in insert mode until you press ESC.

Insert Commands Summary

Match each command to where it enters insert mode: i a I A o O.

i  before the cursor
a  after the cursor
I  beginning (first nonblank) of the line
A  end of the line
o  new line below the cursor
O  new line above the cursor

Numeric Count With Insert

What does 50i* ESC do?

It inserts 50 asterisks. i, I, a, and A accept a numeric prefix that repeats the inserted text after ESC.

Count Exception For o And O

Which insert commands do NOT take a useful numeric prefix?

o and O — unlike i, I, a, and A, they do not usefully repeat with a numeric prefix.

Append At End Of Word

Which command combo appends text at the end of the current word?

ea — e moves to the end of the word, then a appends after it.

Substitute Commands

What do s, S, and R do?

s deletes the character under the cursor and substitutes text (same as c SPACE); S deletes the whole line and substitutes (same as cc); R overstrikes existing characters until ESC.

Join Lines

Which command joins the line below onto the current line?

J (capital) — joins the next line up onto the current line.

Whitespace After Join

How does J adjust whitespace when joining lines?

It leaves two spaces after a period, no space before a ), and one space otherwise, converting the newline into that space.

Join Count

What do 1J, 2J, and 3J do?

Both 1J and 2J join the current line with the one following it. A count of three or more (like 3J) joins that many lines, including the current one.

Join Without A Space

In Vim, how do you join lines without inserting a space between them?

gJ — joins lines and glues them directly with no added space.

J Versus j Pitfall

What is the difference between J and j, and what causes commands to behave as uppercase unexpectedly?

J joins lines; j moves the cursor down. If CAPS LOCK is on, lowercase commands are interpreted as uppercase — turn it off, press ESC, and u to undo.
