Forward By Word

In command mode, which key moves the cursor forward to the start of the next word?

w — moves forward to the beginning of the next word.

Backward By Word

Which command moves the cursor backward to the start of the previous word?

b — moves backward to the beginning of the previous word.

End Of Word

Which command moves to the end of the current (or next) word?

e — moves forward to the end of the word.

Word Versus WORD

What is the difference between a lowercase "word" (w b e) and a capitalized "WORD" (W B E)?

A lowercase word is letters, numbers, and underscores, so punctuation acts as a boundary. A capitalized WORD is separated only by whitespace, so punctuation is not a boundary.

What Counts As A Word

Which characters make up a "word" for the w, b, and e motions?

Letters, numbers, and underscores. Whitespace and punctuation separate words.

Multiply A Word Motion

How do you move forward two words in a single command?

2w — a numeric count before a motion repeats it (e.g., 5B moves back five WORDs).

cw Equals ce

Although w and e are different motions, why do cw and ce behave identically?

Both change text from the cursor to the end of the current word — for the change operator, cw is treated the same as ce.

Sentence Motions

What do ) and ( do?

) moves to the beginning of the next sentence; ( moves to the beginning of the current sentence.

How A Sentence End Is Detected

How does vi (versus Vim) recognize the end of a sentence?

The end is marked by ., !, or ?. Classic vi requires two following spaces (or end of line); Vim needs only a single following space.

Paragraph Motions

What do } and { do?

} moves to the beginning of the next paragraph; { moves to the beginning of the current paragraph.

Definition Of A Paragraph

What defines the boundary of a paragraph for the { and } motions?

Text up to the next blank line, or up to a default troff paragraph macro (.IP, .PP, .LP, .QP).

Section Motions

What do ]] and [[ do?

]] moves to the beginning of the next section; [[ moves to the beginning of the current section.

Zero Versus Caret

What is the difference between 0 and ^?

0 moves to the very first position of the line (even if it is blank). ^ moves to the first nonblank character, skipping leading spaces and tabs.

End Of Line

Which command moves the cursor to the last position of the current line?

$ — moves to the end of the current line.

Move To A Column

How do you move to column n of the current line?

n| — e.g., 5| moves to column 5 (or the end of the line if n is past the last character).

First Nonblank Next And Previous Line

What do + (or ENTER) and - do, and how do they differ from j/k?

+/ENTER move to the first nonblank character of the next line; - moves to the first nonblank of the previous line. Unlike j/k, they jump to the first nonblank character rather than keeping the same column.

Go To Last Line

Which command jumps to the last line of the file?

G — with no count, goes to the last line of the file.

Go To First Line

What are two ways to jump to the first line of the file?

gg (Vim) or 1G.

Go To A Numbered Line

How do you jump to line 44? Give a normal-mode and an ex-command form.

44G in normal mode, or :44 from the ex/colon prompt.

Show Current Line Info

Which command displays the current line number, total lines, and percentage through the file?

CTRL-G — shows file info on the status line (it is not itself a motion).

Bounce Back After A Jump

After jumping with G or a search, how do you return to your previous position?

Two backquotes ` return to the exact previous position; two apostrophes''` return to the start of that line.

Motions As Edit Targets

What do d) and 2y} do?

d) deletes from the cursor to the end of the current sentence. 2y} yanks (copies) two paragraphs ahead.

Editing Grammar Pattern

What is the general pattern for composing an edit from an operator and a motion?

[count] operator [count] motion — e.g., 2yw, d}, c$. The motion tells the operator how much text to act on.
