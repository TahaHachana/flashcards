Two Kinds Of Set Options

What two categories of options can :set change?

Toggle (Boolean) options that are either on or off, and value options that take a numeric or string value.

Turn A Toggle Option On

How do you turn the toggle option ignorecase on?

:set ignorecase (or :set ic).

Turn A Toggle Option Off

How do you turn a toggle option named option off?

:set nooption — prefix the name with no.

Flip A Toggle Option

In Vim, how do you flip a toggle option to the opposite of its current value?

:set option! — append an exclamation mark.

Set A Value Option

How do you set a value option such as wrapmargin to 10?

:set wrapmargin=10 — use an equals sign with no surrounding spaces.

Long Name Versus Abbreviation

What is ic short for, and is :set ic equivalent to :set ignorecase?

ic is the abbreviation for ignorecase; the two commands are equivalent.

List All Options

Which command displays the complete list of options and their current values?

:set all

List Only Changed Options

What does the bare command :set (with no arguments) show?

Only the options you have changed, in your .exrc/.vimrc or during the current session.

Query One Option Value

How do you find the current value of a single option, for example wrapmargin?

:set wrapmargin? — append a question mark to the option name.

laststatus Values

What do the values 0, 1, and 2 of laststatus mean?

0 = never show a status line, 1 = show only if there are at least two windows, 2 = always show one.

incsearch Purpose

What does enabling incsearch do?

It turns on incremental searching, so Vim jumps to matches as you type the search pattern.

number Option

What does :set number do?

It displays line numbers in the left margin of the window.

smartcase Behavior

What does smartcase do, and which option does it depend on?

It overrides ignorecase when the search pattern contains an uppercase character; it is only meaningful when ignorecase is on.

Search Behavior Defaults

Which three default options control case-sensitivity, wrapping, and wildcards in searches?

ignorecase, wrapscan, and magic; reverse them with noignorecase, nowrapscan, and nomagic.

Help For An Option

How do you open the built-in help for an option such as incsearch?

:help 'incsearch' — wrap the option name in single quotes so help knows it is an option.

Prefer Full Names In Config

Should you use option abbreviations or full names in your .vimrc and scripts, and why?

Use full names; configuration files and scripts are easier to read and debug than with cryptic abbreviations.
