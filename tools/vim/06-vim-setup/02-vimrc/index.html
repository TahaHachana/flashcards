Purpose Of vimrc

What is the .vimrc file for?

It permanently configures Vim's editing characteristics — options, global variables, functions, abbreviations, and key mappings — so they apply on every launch.

Comments In vimrc

How do you write a comment in a .vimrc file?

Begin it with a double quote "; everything from the quote to the end of the line is ignored.

Colon Optional In vimrc

In a .vimrc, is set autoindent different from :set autoindent?

No. In a configuration file the leading colon is optional, so the two are identical.

Temporarily Disable A Setting

What is an easy way to temporarily disable one line in your .vimrc without deleting it?

Put a double quote " at the start of that line to comment it out.

Startup Reads First Match

When Vim starts, how many of its initialization sources does it execute?

Only the first one it finds; finding any one stops the search.

Initialization Order

What is the order of Vim's startup initialization sources?

1. VIMINIT environment variable (run as an ex command)
2. User vimrc file ($HOME/.vimrc on Unix; _vimrc on Windows)
3. Local .vimrc/.gvimrc/.exrc in the current directory, only if 'exrc' is set

Unix vimrc Location

Where does Vim look for the user vimrc on Unix and macOS?

In $HOME/.vimrc.

EXINIT Fallback

When is the EXINIT environment variable used?

It is used in place of VIMINIT when VIMINIT is not defined.

MYVIMRC Override

What does setting the MYVIMRC environment variable do?

If it names an existing init file, Vim uses that file and consults no other initialization file.

VIMINIT Multiple Commands

How do you put multiple ex commands in VIMINIT?

Separate them with vertical bars |.

exrc Option Effect

What does setting the exrc option enable?

It makes Vim also read configuration files (.vimrc, .gvimrc, .exrc) from the current directory, allowing per-project config.

Source An Alternate Config

How do you load an alternate settings file on demand from within Vim?

With the source command, e.g. :so .progoptions (:so is short for :source).

What secure Does

What does the secure option do?

It disables certain potentially dangerous commands in the startup configuration file.

When secure Auto Enables

When is secure automatically enabled?

When the .vimrc/.exrc being read is not owned by you, protecting against malicious local config files.

compatible Off Since Vim 8

Since Vim 8.0, when does Vim automatically turn compatible off?

When a runtime or system-wide defaults.vim file exists, so a fresh Vim shows Vim features rather than strict vi behavior.
