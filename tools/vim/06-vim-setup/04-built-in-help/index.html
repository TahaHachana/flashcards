Open The Main Help

Which command opens Vim's main help file?

:help

Jump To A Help Tag

With the cursor on a help tag (such as |bars|), how do you jump to that subject?

Press CTRL-].

Jump Back In Help

After following a help tag, how do you jump back?

Press CTRL-O (repeat to go further back).

Close The Help Window

How do you close just the help window?

:q

Move Within Help

Which keys move the cursor around inside the help window?

The cursor keys, or h j k l.

Help For An Option

How do you ask help about the option textwidth?

:help 'textwidth' — wrap the option name in single quotes.

Help For An Ex Command

How do you ask help about the ex command :quit?

:help :quit — prefix with a colon.

Help Context Prefixes

What help prefixes mark insert-mode, visual-mode, and command-line-editing commands?

i_   insert mode      :help i_<Esc>
v_   visual mode      :help v_u
c_   command-line     :help c_<Del>

Help For Substitute Flags

Which help command shows the flags (like g and c) for the substitute command?

:help s_flags

Help Tab Completion

What happens if you type :help split and press TAB?

Help cycles through matching topics (:split, :vsplit, splitview, and more); press ENTER on the highlighted one to open it.

List Matching Help Entries

After typing :help word, how do you see all matching help entries?

Press CTRL-D.

Search All Help Files

Which command searches across all of Vim's help files for a word?

:helpgrep word

Help Regex Prefix

What prefix do you use to look up a regular-expression item in help, e.g. [?

A forward slash: :help /[

The vimtutor

What is vimtutor?

A roughly 30-minute interactive course that teaches the basic Vim commands.

Read The User Manual

Which help topic begins Vim's user manual, meant to be read from start to end?

:help usr_01.txt
