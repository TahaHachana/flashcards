Define An Abbreviation

How do you define an abbreviation so imrc expands to "International Materials Research Center"?

:ab imrc International Materials Research Center

Abbreviations Are Insert Mode

In which mode do abbreviations defined with :ab expand?

In insert mode.

When An Abbreviation Expands

What triggers an abbreviation to expand?

Typing a non-alphanumeric character after it — punctuation, a space, a carriage return, or ESC — and only when it was typed as a whole word.

Remove And List Abbreviations

How do you remove the abbreviation imrc, and how do you list all abbreviations?

:unab imrc removes it; bare :ab lists all current abbreviations.

No Tail Recursion

Why is :ab PG This movie is rated PG rejected?

The abbreviation cannot appear at the end of its own expansion — that would loop infinitely; Vim reports "No tail recursion."

The map Command

What does :map x sequence do, and in which mode?

It binds the key x to a sequence of editing commands in command (normal) mode.

Unmap And List Maps

How do you remove the map for x, and how do you list all maps?

:unmap x removes it; bare :map lists currently mapped keys.

Keys Free To Map

Which letters does vi leave unused in command mode and therefore safe to map?

g, K, q, V, and v (plus control keys ^A, ^K, ^O, ^W, ^X and symbols _, *, \, =).

Insert Mode Map

How do you make a map apply in insert mode instead of command mode?

Add an exclamation mark: :map! x sequence (remove it with :unmap! x).

Escaping Special Keys

How do you include keys like ENTER or ESC inside a map definition?

Escape them with CTRL-V; ENTER then shows as ^M and ESC as ^[.

Carriage Return In Map

How does a carriage return appear on screen inside a map, and how do you type it?

It appears as ^M, typed as CTRL-V followed by ENTER.

Vertical Bar Restriction

Why can't you use a vertical bar | in an insert-mode map?

| always has special meaning as a separator of multiple ex commands and cannot be escaped in this context.

Default mapleader

What is the default value of mapleader?

A backslash \.

Map With A Leader

How do you map \q to quit Vim using the leader?

:map <leader>q :q<cr>

Change The Leader

How do you change mapleader to a different character such as X?

:let mapleader="X"

noremap Purpose

What does :noremap do differently from :map?

It creates a non-recursive map: the right-hand side is taken literally and not re-expanded through other mappings.

Map A Function Key

What syntax maps function key F1 to a command sequence?

:map #1 commands — #n refers to function key number n.
